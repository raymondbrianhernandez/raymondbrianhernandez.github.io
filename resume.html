CAPSTONE/ELECTIVE PROJECTS

  • Graduate Thesis: Supervisory Control and Data Acquisition (SCADA) Implementation on a Power Plant

    • 2018

    • Research is focused on assessing the impact of Supervisory Control and Data Acquisition (SCADA) system implementation on the military base Camp Lemonnier located in Djibouti, Africa. The model accounts for technical parameters, such as types of SCADA and control systems, SCADA and control system components, and SCADA communication protocols.

  • Capstone Project (Individual): Discrete Operational Amplifier

    • 2017

    • Designed an amplifier with frequency response to DC - 10MHz + 2 dB and gain of 1,000 to 1,500. Used PSPICE to design, simulate, and test the circuit using MOSFETs, BJTs, and other discrete components. The actual circuit was soldered on a through-hole PCB.

  • Capstone Project (Group): Analog to Digital Signal Converter Application in an Audio Recorder

    • 2017

    • Designed an audio recorder for both lines in audio signals and microphone signals that enables to be recorded in an SD card at 44.1 kHz sample rate using 16-bit samples. Pre-amplifier and output amplifier are designed using PSPICE using 741 op-amp chip, discreet components, and hardware for playback/recording. Primarily used an Arduino MEGA board and programming using C/C++. Surface mount soldering on a PCB.

  • Elective Project: Alcohol Detector Breathalyzer

    • 2017

    • Created a device to detect alcohol content in the breath of drivers under the influence of alcohol. The circuit is designed using MQ-135 module connected to Digilent Zybo Zynq 7000 SoC development board. Board is coded in C and VHDL.

  • Elective Project: Pocket Uni App for Mobile Devices

    • 2016

    • Created an app for Android devices to assist students in their academics. It features an assignment manager, messaging service, and a market like Craigslist. Students need to use their CSUN email to sign-up for security. Coded in Javascript using Android Studio. Registered users are managed through a MySQL database.
